# Weather Checker Web App

## Description
A simple web application built using Express.js, Axios, and EJS that tells users whether it will rain tomorrow in a chosen city.

## Installation
1. Clone the repository
2. Run `npm install`
3. Start server using `npm run dev`

## Technologies Used
- Node.js
- Express.js
- Axios
- EJS
- Open-Meteo API

## How to Run
Visit: http://localhost:3000
